Detailed Description

This Controller smply takes the 'content' configuration and outputs it. Period. This is mostly for testing reasons.

Okay, there is one secret: all other configuration settings can be used in the template via {curly} placeholders. You need to set the placeholders[] setting with every placeholder you want to use.
