\subsection{Detailed Description}
In the first place, the application holds the Request\+Handler Instance and serves the start methode \hyperlink{classWEPPO_1_1Application_1_1Application_a46df3113be7bf1f246c00a12dd9b1e3f}{run()}. By default, \hyperlink{classWEPPO_1_1Application_1_1Application_a46df3113be7bf1f246c00a12dd9b1e3f}{run()} will call \hyperlink{classWEPPO_1_1Application_1_1Application_a1a0c0171a086ce9a74fdb58ab891a488}{auto\+Request()} to handle a normal Request. It will also catch redirect exceptions.
