\subsection{Detailed Description}
\hyperlink{classWEPPO_1_1Helpers_1_1MultiSession}{Multi\+Session}

Diese Klasse ermöglicht es, mehrere getrennte P\+H\+P-\/\+Sessions zu verwalten. Die Session-\/\+Variablen der einzelnen Sessions sind getrennt von einander. Die Funktionalität setzt auf die session\+\_\+$\ast$() Funktionen von P\+HP auf und ermöglicht über wechseln von Session-\/\+Id und Session-\/\+Name, das Auslesen und Manipulieren mehrerer Sessions. Bei jeder Operation wird die gewünschte Session geöffnet und danach wieder geschlossen (nicht beendet!).

Eine Session kann gezielt gestartet und beendet werden. Jede \hyperlink{classWEPPO_1_1Helpers_1_1MultiSession}{Multi\+Session} hat dabei ihren eigenen Cookie.

Multi\+Sessions können nicht parallel mit manuellen session\+\_\+start() Aufrufen verwendet werden.
