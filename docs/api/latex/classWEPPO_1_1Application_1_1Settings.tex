\subsection{Detailed Description}
Manages globally available application settings.

It implements the singelton pattern so that there is only one settings object throughout the application. It initially loads a file called \hyperlink{default_8php}{default.\+php} in the \hyperlink{namespaceWEPPO}{W\+E\+P\+PO} directory. With \hyperlink{classWEPPO_1_1Application_1_1Settings_a5fdde58045099bacbe6a39eb9232f48c}{load\+Settings()} you can load application specific settings and thereby overwrite the defaults.
